★ kill feature

Rollback wrangler doesn't have

Cloudflare offers no CLI rollback for Pages. nt-deploy archives every push locally and lets you restore any version in seconds.

  • nt-snapshots client — the deploy history
  • nt-rollback client — back to the previous one
  • nt-rollback client 1700000000 — to an exact version
$ nt-snapshots client
  1719500000  2026-06-15 11:40  18K  (current)
  1719490000  2026-06-15 10:02  18K  (rollback →)
  1719400000  2026-06-14 22:15  17K
$ nt-rollback client
  ⏪ Rolling back to snapshot from 2026-06-15 10:02
  ✅ Rollback complete
